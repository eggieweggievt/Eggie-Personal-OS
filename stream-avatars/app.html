📖 How it all works

🖥️ Putting it in OBS

  1. Sources → + → Browser
  2. Paste your link in URL (don't tick "Local file")
  3. Width 1920 · Height 1080 — must match your canvas
  4. FPS: Custom 60 (30 is fine)
  5. Untick "Shutdown source when not visible"
Looks empty? Normal — avatars appear when people chat. Add &debug=1 while arranging.

🔧 If something's weird

Squished/stretched: set the OBS source to 1920×1080 and use right-click → Transform → Fit to screen (don't drag-resize the box). Add &debug=1 — if the green "100×100 square?" box looks like a rectangle, that's the cause.
Old version showing: right-click source → Refresh cache of current page.
